# covidImportation



## Installation

Required package:
```{r}
devtools::install_github("HopkinsIDD/globaltoolboxlite")
```

Current version:
```{r}
devtools::install_github("HopkinsIDD/covidImportation")
```

Stable frozen version:
```{r}
devtools::install_github("HopkinsIDD/covidImportation", ref="v1.6.1")
```
